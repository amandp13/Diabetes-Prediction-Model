# Diabetes Prediction Model.
- Diabetes Prediction using Machine Learning in Apache Spark.
- A Logistic Regression Model to predict the diabetes from the kaggle Diabetes Dataset.
- Using Pyspark MLlib.

## Requirements.
- Requires Pyspark dependency.
